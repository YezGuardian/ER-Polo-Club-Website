# East Rand Polo Club Website

The official website for East Rand Polo Club, a premier equestrian sanctuary with over 80 years of sporting excellence.

**Built by White Paper Concepts (Pty) LTD.**

## Overview

This modern, high-performance web application showcases the heritage, facilities, and sporting events of the East Rand Polo Club. It features an immersive user experience with smooth animations, interactive galleries, and a responsive design that adapts seamlessly to all devices.

## Key Features

-   **Immersive Visuals**: Full-screen video hero sections with smooth autoplay and placeholder transitions.
-   **Day/Night Mode**: Fully supported light and dark themes with persistent user preference.
-   **Interactive Galleries**:
    -   **Masonry Layout**: Dynamic image grids for the Gallery and Venue pages.
    -   **Lightbox Modals**: Full-screen image viewing with keyboard navigation and touch support.
    -   **Carousels**: Smooth sliding image carousels for the Academy page.
-   **Rich Animations**:
    -   **Scroll Reveal**: Elements fade and slide in as the user scrolls.
    -   **Page Transitions**: Smooth entrance animations for a polished feel.
    -   **Hover Effects**: Interactive states for cards, buttons, and images.
-   **Comprehensive Content**:
    -   **Heritage**: Timeline and historical narrative of the club.
    -   **Academy**: Information on polo lessons and training.
    -   **Tournaments**: Fixtures and event details.
    -   **Venue**: Venue hire information with gallery.
    -   **Contact**: Integrated map and contact details.

## Technical Architecture

The project is built using a modern frontend stack focused on performance, type safety, and developer experience.

### Core Technologies
-   **Framework**: [React 18](https://reactjs.org/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Routing**: [React Router DOM](https://reactrouter.com/)

### Styling & UI
-   **CSS Framework**: [Tailwind CSS](https://tailwindcss.com/) for utility-first styling.
-   **Animations**: [Framer Motion](https://www.framer.com/motion/) for complex animations and gestures.
-   **Icons**: [Lucide React](https://lucide.dev/) for consistent, lightweight iconography.

### Project Structure
-   `/components`: Reusable UI components (Hero, Navbar, Footer, Carousel, etc.).
-   `/pages`: Route components representing each page of the website.
-   `/public`: Static assets (images, videos, icons).
-   `/src`: Entry point and global styles.

## Getting Started

To run this project locally:

1.  **Prerequisites**: Ensure you have Node.js installed.
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Start Development Server**:
    ```bash
    npm run dev
    ```
4.  **Build for Production**:
    ```bash
    npm run build
    ```
